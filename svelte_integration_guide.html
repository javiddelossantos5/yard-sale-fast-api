🎨 Svelte Integration Guide

✅ Step 1: Copy Svelte Files

Run these commands on your local machine:

# Copy Svelte frontend to server scp -r /Users/javiddelossantos/Documents/Github/Strata/svelte-yard-sale/* javiddelossantos@10.1.2.165:/home/javiddelossantos/yard-sale-svelte/ # Copy integration script to server scp /Users/javiddelossantos/Documents/Github/Strata/fast-api-test/server_svelte_integration.sh javiddelossantos@10.1.2.165:/home/javiddelossantos/fast-api-test/

✅ Step 2: SSH to Server

Connect to your server:

ssh javiddelossantos@10.1.2.165

✅ Step 3: Run Integration Script

On the server, run these commands:

cd /home/javiddelossantos/fast-api-test chmod +x server_svelte_integration.sh ./server_svelte_integration.sh

🎉 Expected Results

After running the script, you should see:

  • ✅ Svelte Frontend: http://10.1.2.165/
  • ✅ FastAPI Backend: http://10.1.2.165:8000/
  • ✅ API Docs: http://10.1.2.165:8000/docs

🔍 Troubleshooting

If something goes wrong, try these commands on the server:

# Check service status docker-compose ps # View logs docker-compose logs -f # Restart services docker-compose restart # Stop all services docker-compose down

🌐 Domain Configuration

Once everything is working on IP, configure your domain:

  • Domain: garage.javidscript.com
  • Forward to: 10.1.2.165:80
  • SSL: Enable in Nginx Proxy Manager
